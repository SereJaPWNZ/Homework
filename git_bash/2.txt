1. На локальном репозитории сделать ветки для:
- Postman
- Jmeter
- CheckLists
- Bag Reports
- SQL
- Charles
- Mobile testing

git checkout -b Postman                                                     /// создать и перейти в новую ветку с именем "Postman"
git branch      Jmeter CheckLists Bag_Reports SQL Charles Mobile_testing    /// создать ветки с именами "Jmeter", "CheckLists", "Bag_Reports", "SQL", "Charles", "Mobile_testing"


2. Запушить все ветки на внешний репозиторий
git push -u origin Bag_Reports Charles CheckLists Jmeter Mobile_testing SQL
