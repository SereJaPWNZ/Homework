Первое ДЗ 
Linux terminal (GitBash) commands

1) Посмотреть где я
	command: pwd
2) Создать папку
	command: mkdir test
3) Зайти в папку
	command: cd test
4) Создать 3 папки
	command: mkdir test22 test33 test44
5) Зайти в любоую папку
	command: cd ./dsd/sdsd/fddfe
6) Создать 5 файлов (3 txt, 2 json)
	command: touch file_txt{1..3}.txt file_json{1..2}.json
7) Создать 3 папки
	command: mkdir folder{1..3}
8) Вывести список содержимого папки
	command: ls -la
9) + Открыть любой txt файл
	command: vim test_4.txt
10) + написать туда что-нибудь, любой текст.
	command: i (дает возможность редактировать документ)
	command: napisat luboy tekst (вводимый текст)
11) + сохранить и выйти.
	command: : esc (выйти из режима редактирования документа)
	command: :wq (сохранить и выйти)
12) Выйти из папки на уровень выше
	command: cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку.
	command: mv file_txt1.txt file_txt2.txt folder1
14) скопировать любые 2 файла, которые вы создали, в любую другую папку.
	command: cp file_json1.json file_json2.json folder2
15) Найти файл по имени
	command: find git_bash/ -name file_txt3.txt
	result: git_bash/file_txt3.txt
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
	// "команда grep" означает обнаружение файла с конкретным обозначением.
	// просто посмотреть последие 10 строк документа с помощью "tail"
	command: tail test_4.txt
17) вывести несколько первых строк из текстового файла
	command: head -n3 test_4.txt 						//вывожу 3 первых строки
18) вывести несколько последних строк из текстового файла
	command: tail -n3 /var/log/syslog 					//вывожу 3 последних строки
19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
	//Команда less больше подходит для чтения текстовых файлов, не загромождая экран. 
	Его также можно использовать для чтения файлов с изменениями в реальном времени.
	Эта опция +Fпозволяет меньше отслеживать изменения, внесенные в текстовый файл.
	command: less +F test_4.txt
20) вывести дату и время
	command: date

узнать местоположение в файловой системе: pwd
создать папку/несколько папок: mkdir dir    //// mkdir dir1 dir2 dir3 /// mkdir{1..3}
создать файл/несколько файлов: touch название_файла.формат_файла //// touch название_файлов{1..3}.формат_файлов
перейти в каталог выше: cd ..
перейти в корень: cd
перейти на два каталога выше: cd ../..
отрыть и редактировать файл: vim название_файла
просмотреть файл в терминале: cat название_файла
